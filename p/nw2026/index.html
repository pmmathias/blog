// Mobile First?

60% der Leser kommen mobil. Das sehen sie:

  • Logo wird zu „Nord Wi..." abgeschnitten
  • Navigation scrollt horizontal ins Nichts
  • Kein Hamburger-Menü

  • Content ohne sichtbare Struktur
Das bedeutet
Jeder zweite Besucher sieht eine kaputte Seite. Partner wie Commerzbank und HanseMerkur werden neben einem kaputten Layout präsentiert.

// Umsatzpotenzial

912 Artikelseiten — ohne eine einzige Anzeige

Aktuell werden Anzeigen nur auf der Startseite platziert — 7 Werbepartner in der Sidebar. Auf den 912 Artikelseiten: nichts. Das ist verschenktes Potenzial.

  • Banner in Sidebar + Inline im Artikel
  • Advertorials mit Presserecht-Kennzeichnung
  • Impressions- und Klick-Tracking
  • Reports für Werbekunden (PDF/CSV)
